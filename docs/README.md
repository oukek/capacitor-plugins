## 概述

Capacitor 插件系统允许你扩展和增强应用程序的原生功能。通过创建自定义插件，你可以：

- 访问原生 API
- 在 Web 代码和原生代码之间建立桥接
- 实现特定平台的功能
- 重用和共享原生功能
